# ネタトレンドデイリー ワークフロー（完全自動化版）

**このワークフローを実行すると、AIが自動的に:**
1. Web検索で最新記事を30本収集
2. りゅう専用フィルターで10本に厳選
3. 読まない方がいい5本も辛口選定
4. Obsidian用マークダウンを自動生成・保存

**あなたがやること:** 確認して、自分の判断と次のアクションを追記するだけ！

---

## 実行手順

### 1. Web検索で候補記事を収集（AI実行）

以下の条件で検索を実行:

**対象期間:** 過去24時間（必要なら48時間まで拡張）

**重点テーマ（優先度順）:**
1. 科学コミュニケーション（教育/オウンドメディア/編集/メディアリレーション含む）
2. 最新AIトレンド（生成AI、研究、プロダクト、政策/倫理、活用事例）
3. 健康・Well-being（習慣化、睡眠、運動、メンタル、行動科学）
4. 趣味: サウナ / 映画 / 音楽 / テニス / 漫画 / Museum探訪 / お酒

**情報源の方針:**
- ✅ 一次情報（論文、公式ブログ、官公庁、企業の一次発表）を優先
- ❌ まとめサイト/扇情的/根拠薄は原則落選
- 同じ話題の重複は1本に統合

// turbo
```bash
# AIが自動実行: 各テーマで検索クエリを実行
# 例: "科学コミュニケーション OR science communication site:nature.com OR site:science.org after:2026-02-08"
```

### 2. ベスト10を厳選 ＆ 読まない5本を選定（AI実行）

収集した候補（約30本）から、以下の基準で10本に厳選し、かつ「読まない方がいい」5本を特定する。

**選定基準:**
- りゅうに刺さる「読む価値」があるか
- マネタイズの種 `[Monetize]` または 純粋な興味 `[Interest]` が明確か
- 一次情報か、信頼できるソースか

**各記事に付与:**
- **Hook（煽り文）**: クリック判断のための1〜2文（要約禁止）
- **Why**: なぜりゅうが読むべきか（`[Monetize]`/`[Interest]` + 一言）
- **期待リターン**: 短く
- **推奨アクション**: Read / Save / Skip

### 3. リンクの有効性・正確性検証（AI実行：最重要）

選定した10本+αについて、以下のツールを **必ず事前に使用** して検証すること。

- **生存確認**: `read_url_content` または `browser_subagent` を使用し、リンクが 404/403 等のエラーにならないか、およびドメインのトップページではなく「個別の記事ページ」であるかを確認する。
- **内容の整合性**: 記事のタイトルやリード文を確認し、ステップ2で作成した「Hook」や「Why」の内容がハルシネーション（嘘）でないことを確認する。
- **NGパターンの排除**: 以下の場合はその記事を候補から外し、次点の有効な記事に差し替えること。
    - リンク切れ（404等）
    - 有料ウォールの裏側で内容が全く確認できない
    - ドメイン（トップページ）へのリンクしか取得できない
    - 内容がテーマから逸脱している

### 4. Obsidian用マークダウンを生成・保存（AI実行）

**生成時の最終ルール:**
- **検証済みリンクのみ掲載**: ステップ3で正常にアクセスと内容が確認できたリンクのみを掲載する。
- **ハルシネーションの完全排除**: 内容の整合性が取れない情報は一切含めない。
- **一次ソース優先**: 可能な限り公式発表、論文、一次メディアの直接URLを使用する。

### 5. ユーザーが確認・追記

生成されたファイルを開いて:
1. ベスト10を確認
2. 「自分の判断」セクションを埋める
3. 「次の1アクション」を1つだけ決める
4. 実行！

## カスタマイズのヒント

### 収集ソースを固定化する

`life/prompts/neta-trend-daily.md` の「情報源の方針」セクションに追加:

**必ず見るべきソース（例）:**
- Nature, Science, arXiv（科学）
- OpenAI Blog, Anthropic Blog, Google AI Blog（AI）
- 特定のニュースレター（例: The Batch, Import AI）

**除外すべきソース（例）:**
- まとめサイト、信頼性の低いメディア

### 重点テーマの調整

上記「重点テーマ」の優先度や内容を変更可能。
